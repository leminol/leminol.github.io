Wavy Navy

An epic naval game by leminol.

Wavy Navy is a game that ship and cannons are separated. It's a survival that you' re trying to get the highest score.

NOTE: Tab key for releasing in Browsers crashes the game. DO NOT USE IT! Right click :)

This game is built for GDevelop Game Jam 3.
